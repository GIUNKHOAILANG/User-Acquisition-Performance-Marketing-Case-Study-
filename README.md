# User-Acquisition-Performance-Marketing-Case-Study-
User acquisition case study for Cube Escape Collection (Rusty Lake) - Q1 2026 campaign analysis across Unity Ads, ironSource, AppLovin, Meta, and TikTok in 3 region tiers. Creative fatigue detection, CPM/CPI/ROAS analysis, ASO audit, and a 10-point prescriptive action plan.


**Cube Escape Collection** is a narrative puzzle/escape-room game by Rusty Lake. Comprising of 9 chapters, 5M+ downloads, hybrid monetisation (rewarded ads + IAP hint packs). Unlike common hyper-casual titles, this mobile game has 12–18 minute sessions and finite content mean LTV has a natural ceiling, creative strategy must match the surreal art style, and broad targeting wastes spend on users who churn in seconds. 

This case study analyses Q1 2026 UA performance across 36,839 daily data points spanning Unity Ads, ironSource, AppLovin, Meta, and TikTok in T1 (US/UK/DE), T2 (BR/MX/PL), and T3 (IN/ID/PH) using a generated dataset. 

The findings identify a 29% creative fatigue-driven IPM decline, a T2 Brazil efficiency opportunity at $0.52 CPI / 8.1% D7 ROAS.
With the dashboard in Power BI, I managed to produce a 10-point prescriptive action plan projecting 16% CPI reduction and 35% D7 ROAS improvement for Q2.
